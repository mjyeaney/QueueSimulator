Queueing Simulation

Description

This report details the result of a discrete event simulation against a single-queue processing system. The following parameters describe the system:

  • Arrival rate: Poisson, λ={{ARRIVAL_RATE}}
  • Service rate: μ={{SERVICE_RATE}}
  • Total arrivals (requests): {{TOTAL_CUSTOMERS}}
  • Simulation time: {{SIMULATION_TIME}}
  • Total test duration: {{TOTAL_DURATION}}
  • Recovery duration: {{RECOVERY_DURATION}}
  • Mean queue length: {{MEAN_QUEUE_LENGTH}}
  • Mean wait time: {{MEAN_WAIT_TIME}}
  • Mean utilization: {{MEAN_UTILIZATION}}

Notes

The simulation stops new arrivals after 2 hours; the system is then given time to drain any residual customers.

Note that utilization reports values > 100%; this is only for illustrative purposes.
